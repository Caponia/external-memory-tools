# 从这里开始

你不需要记住目录结构，只需按下面的情况选择。

## 只是问问题

直接向 AI 提问。需要个人偏好或项目背景时，让它先读取：

```text
{{MEMORY_PATH}}/README.md
```

## 要完成具体工作

在 `Workspaces/Tasks/` 新建独立目录保存输入、输出和过程记录。长期记忆只作为只读背景，不要把所有临时过程写回记忆库。

## 要维护长期项目

在 `Workspaces/Projects/` 建立项目目录；项目细节留在项目工作区，只把跨任务长期需要的状态摘要写入 `Memory/PROJECTS.md`。

## 想更新长期记忆

1. 先让 AI 在聊天中提出候选内容，不写文件。
2. 确认后，明确说“写入 INBOX”。
3. 再次检查候选内容。
4. 只有再次明确批准，才合并到主文件并更新 `CHANGELOG.md`。

## 最简单的记法

> 记忆是一份真源；默认只读；更新先走候选区。
